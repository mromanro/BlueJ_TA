Directory Structure

src
├───+libs/                      Contains required libraries.
├───doc/                        Documentation. Generated by running "ant doc".
│   └───index.html              Documentaion main.
└───Extension/                  The main package.
    ├───BackEnd/                The package used to load and run exercises.
    │   └───runner/             This folder is hidden from BlueJ, but,
    │                            compiled by the build and used to run test
    │                            units on the user's solutions.
    ├───exercises/              This folder contains the default exercise
    │                            files.
    └───GUI/                    This folder contains JavaFx GUI sources.
